## 개요
~/Downloads/zap.jpg 파일에서 흰색 배경을 제거하고 투명 PNG로 변환하여 프로젝트 아이콘으로 사용

## 작업 내용
1. 흰색 배경 제거 (투명화)
2. 1024x1024로 스케일 업
3. 다중 크기 아이콘 생성 (128, 256, 512, 1024px)
4. assets/icons/ 디렉토리에 저장

## 출력 파일
- assets/icons/zap-128.png
- assets/icons/zap-256.png
- assets/icons/zap-512.png
- assets/icons/zap-1024.png

## 목적
써드파티 zap 관련 도구에서 아이콘으로 활용할 수 있도록 제공
